Kinodex is an archival assistant specialized for physical movie collections. I built this because Letterboxd does not have adequate features for cataloging physical media and many existing apps lock away basic functionality behind paywalls. With Kinodex, you can track every detail of your collection, search your database with barcode scanning, and recieve comprehensive analytics.

Current features include:
- Bulk editing
- Advanced filtering
- Barcode scanning
- Price tracking
- Personal collections/shelves
- Shelf view (experimental)
- Collection list tracking
- CSV import/export
