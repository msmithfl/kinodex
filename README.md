Kinodex is a movie collection assistant specialized for physical media collectors. I built this because Letterboxd does not have adequate features for cataloging physical collections and many existing apps lock away basic functionality behind paywalls. With Kinodex, you can track every detail of your collection, search your database with barcode scanning, and receive comprehensive analytics.

Current features include:
- Bulk editing
- Advanced filtering
- Barcode scanning
- Price tracking
- Personal collections/shelves
- Shelf view (experimental)
- Collection list tracking
- CSV export/import
